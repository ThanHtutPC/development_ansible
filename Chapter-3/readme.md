## Automation Controller
```
Red Hat Ansible Automation Platform 2 includes a component called automation controller.
Called Ansible Tower in earlier versions of Ansible Automation Platform.
- centralized web UI for playbook management
- role-based access control (RBAC)
- centralized logging and auditing

Automation Controller
----------------------------------------------
- Creating Credential Resources
    - Ansible Galaxy/Automation Hub API Token
    - Container Registry
    - GitHub Personal Access Token
    - Machine
    - Vault
    - Source Control
- Creating Project Resources
- Creating Inventory Resources
    - Manually Adding Groups and Hosts 
    - Populating Groups and Hosts by Using a Project Inventory File (Source inventory)
- Creating Job Template Resources
- Launching and Reviewing Jobs
